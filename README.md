# About This Repository
This repository contains my day-wise backend development tasks and practice work.
Each folder represents the work completed on a specific day while learning backend development concepts and building APIs.

# Each Day-X folder may include:
-Server setup files
-Express routes
-Controllers and models
-Database connection logic
-Practice APIs and mini tasks

# Tech Stack Used
-Node.js
-Express.js
-MongoDB
-Mongoose
-dotenv
-npm
-Git & GitHub

# How to Run a Day’s Backend Project
1.Navigate to the specific day folder : cd Day-X
2.Install dependencies : npm install
3.Create a .env file : MONGO_URI=your_mongodb_connection_string
4.Start the server :npm run dev or node server.js
